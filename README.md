# 🕊️ NayePankh Foundation Website

A modern, responsive NGO website built using HTML, CSS, and JavaScript to showcase the mission, impact, programs, and community initiatives of NayePankh Foundation.

## 🌟 Features

- Responsive Design
- Modern NGO-Themed UI
- Dark Mode Toggle
- Smooth Scroll Navigation
- Mobile-Friendly Navigation Menu
- Scroll Reveal Animations
- Interactive Program Cards
- Contact Form
- Impact Statistics Section
- Volunteer & Donation Call-to-Action
- Clean and Accessible Design

### Clone the Repository


### Open Project

```bash

```

### Run Website

Simply open:

```text
index.html
```
